# Firefox Power User Configuration

Opinionated Firefox configuration for power users.

## Goals
- Disable Firefox password manager (use Bitwarden)
- Reduce UI noise
- Improve privacy defaults
- Zero-click setup on new machines

## Usage

1. Install Firefox
2. Locate your Firefox profile directory
3. Copy `user.js` into the profile folder
4. Restart Firefox

Firefox will reapply these settings on every launch.
